# Plataforma Web de Apoyo para Personas con Alzheimer y Cuidadores

## Descripción del Proyecto

Este proyecto consiste en el desarrollo de una plataforma web diseñada para facilitar el seguimiento y apoyo diario de personas que padecen Alzheimer, así como mejorar la comunicación y coordinación con sus cuidadores o familiares.

La aplicación permite gestionar recordatorios, realizar ejercicios cognitivos y mantener un seguimiento del estado del paciente en tiempo real, ayudando a mejorar su autonomía y calidad de vida.

El proyecto ha sido desarrollado como **Proyecto Intermodular (TFG) del ciclo formativo de Administración de Sistemas Informáticos en Red (ASIR)**.

---

## Objetivo

El objetivo principal de la plataforma es ofrecer una herramienta accesible que permita:

* Ayudar a los pacientes a mantener rutinas diarias.
* Facilitar el seguimiento del estado del paciente por parte de familiares o cuidadores.
* Proporcionar ejercicios cognitivos que estimulen la memoria y las capacidades mentales.
* Centralizar información relevante sobre el cuidado del paciente.

---

## Funcionalidades Principales

### Sistema de perfiles

La plataforma cuenta con dos tipos de usuario:

**Paciente**

* Acceso a recordatorios diarios
* Acceso a ejercicios cognitivos
* Interfaz simplificada para facilitar el uso

**Cuidador / Familiar**

* Gestión del perfil del paciente
* Creación y gestión de recordatorios
* Seguimiento del progreso del paciente
* Visualización de la actividad del usuario

---

### Recordatorios

El sistema permite crear recordatorios para ayudar al paciente a mantener rutinas como:

* Toma de medicación
* Citas médicas
* Actividades diarias
* Tareas importantes

---

### Ejercicios Cognitivos

La plataforma incluye ejercicios diseñados para estimular capacidades cognitivas como:

* Memoria
* Atención
* Reconocimiento
* Asociación

---

### Seguimiento del Paciente

Los cuidadores pueden consultar información relacionada con la actividad del paciente dentro de la plataforma para facilitar su seguimiento.
